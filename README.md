# ASCII Art Toy

브라우저에서 바로 실행되는 인터랙티브 ASCII 아트 생성기입니다.

## 실행 방법

`index.html`을 브라우저에서 열면 바로 사용할 수 있습니다. 별도 설치 불필요.

## 기능

- **6가지 패턴**: sine, ripple, spiral, noise, diamond, tunnel
- **8가지 색상 팔레트**: green, cyan, purple, fire, pink, gold, ice, mono
- **5가지 글자 세트**: blocks, braille, classic, katakana, symbols
- **프리셋**: 매트릭스, 파도, 불꽃, 우주, 글리치
- **마우스 인터랙션**: 마우스 위치에 따라 패턴이 반응
- **PNG 저장**: 현재 화면을 PNG로 다운로드

## 컨트롤

| 슬라이더 | 설명 |
|---|---|
| 패턴 | 애니메이션 패턴 종류 선택 |
| 주파수 | 패턴 반복 주기 |
| 속도 | 애니메이션 속도 |
| 밀도 | 글자 밀도 |
| 글자 세트 | 사용할 문자 종류 |
| 마우스 영향 | 마우스 위치가 패턴에 미치는 영향도 |
